# Planning Roadmap

## MVP objective

Launch a working Arabic RTL SaaS-ready shortage workflow for one or more companies/pharmacies with secure roles and clean UX.

## Release sequence

1. Documentation foundation.
2. Technical foundation.
3. Database/security foundation.
4. Auth and profiles.
5. Item master data.
6. Shortage workflow.
7. Admin operations.
8. Testing/deployment.
9. SaaS hardening.
