# Phase 03 — Auth and Roles

## Objective

Implement login, profiles, protected routes, and role-based navigation.

## Deliverables

- Login page.
- Supabase auth integration.
- Profile loading.
- Protected routes.
- Role-aware navigation.

## Exit criteria

- Users can log in/out.
- Inactive/missing profiles are blocked.
- Each role sees appropriate navigation.
