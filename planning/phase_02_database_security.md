# Phase 02 — Database and Security Foundation

## Objective

Build PostgreSQL schema, indexes, RLS policies, and workflow functions.

## Deliverables

- Supabase migrations.
- Tables and indexes.
- RLS policies.
- Workflow functions.
- Seed data.

## Exit criteria

- Cross-tenant data access is blocked.
- Role rules are testable.
- Workflow status changes create audit history.
