# Design System

## Brand direction

Clean, modern, calm, trustworthy pharmacy operations app.

## Language

Arabic RTL UI.

## Font

Recommended MVP font: Tajawal.

## Colors

Suggested palette:

- Primary: teal or blue.
- Background: white / slate-50.
- Text: slate-900.
- Muted text: slate-500.
- Border: slate-200.
- Success: green.
- Warning: amber.
- Danger: red.

## Components

- Cards for mobile request views.
- Tables for desktop request views.
- Badges for statuses.
- Dialog/sheet for add request.
- Toasts for feedback.

## Spacing

- Use generous spacing.
- Touch targets at least 44px height.
- Avoid dense ERP-style screens for MVP.
